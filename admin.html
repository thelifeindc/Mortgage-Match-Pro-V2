Google Sheets Integration

Set up your form to submit entries to a Google Sheet.

Enter the URL of your deployed Google Apps Script web app.
Logs submission data and responses to the browser console.

Instructions

  1. Create a Google Sheet to store form submissions
  2. Create a Google Apps Script web app by going to Extensions > Apps Script in your Google Sheet
  3. Copy and paste the Apps Script code from the instructions document
  4. Deploy the script as a web app and copy the Web App URL
  5. Paste the URL above and click "Save Configuration"
  6. Click "Test Connection" to verify your setup
