Mortgage Match Pro API Demo

This page demonstrates how to use the Mortgage Match Pro API to search for homebuyer programs across Maryland.

API Examples

Get All Programs

GET /api/programs

Search Programs by Criteria

POST /api/programs/search
{
  "county": "montgomery",
  "firstTimeBuyer": true,
  "creditScore": 640
}

Get Available Counties

GET /api/counties
